# Code of Conduct

## Our Standards

We expect all contributors to:

- Be respectful and professional in all interactions
- Focus on technical merit and constructive feedback
- Welcome newcomers and help them get started
- Respect differing viewpoints and experiences
- Accept constructive criticism gracefully

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or personal attacks
- Trolling, insulting comments, or inflammatory language
- Publishing others' private information without permission
- Any conduct that creates an intimidating or hostile environment

## Enforcement

Project maintainers have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

Violations can be reported to ali@jakvan.io. All complaints will be reviewed and investigated promptly.

## Attribution

This Code of Conduct is adapted from standard open source community guidelines.
